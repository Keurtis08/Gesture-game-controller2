About

The project, the team, and the technologies behind it.

The project

Gesture Game Controller was built during Arduino Days 2026 at Semè City, Bénin. The challenge was to create an innovative interface between the human body and a video game, using embedded artificial intelligence running directly on a microcontroller.

🎯
Challenge objective

Build a gesture-based game controller using an Arduino Nano 33 BLE Sense, an Edge Impulse AI model, and a Python script — without modifying the game itself.
