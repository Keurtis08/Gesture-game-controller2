# Tutorials — Overview

These tutorials walk you through setting up and running the project from scratch.

| Tutorial | Estimated time | Description |
|----------|---------------|-------------|
| [Install Arduino IDE](arduino-ide.md) | 10 min | Set up the development environment |
| [Train the AI model](edge-impulse.md) | 30–60 min | Create and export the model on Edge Impulse |
| [Run the project](run.md) | 15 min | Upload the code and launch the demo |

!!! tip "Where to start?"
    If you are starting from scratch, follow the tutorials in this order: **Install Arduino IDE** → **Train the AI model** → **Run the project**.
