Hardware

Components list and wiring diagram for the Gesture Game Controller.

Components list

ComponentQtyRole
Arduino Nano 33 BLE Sense1IMU sensor + AI model + BLE transmitter
USB cable (for Arduino)1Programming only (before the demo)
Powerbank1Wireless power supply during gameplay
PC with Bluetooth1Runs Python script and the game
✅
No extra bridge needed

The Arduino Nano 33 BLE Sense has a built-in BLE 5.0 chip. Most modern laptops have a built-in Bluetooth adapter. They communicate directly — no additional hardware required between them.

Arduino Nano 33 BLE Sense — specifications

Wiring diagram

The setup is entirely wireless during gameplay. The only physical connection is the Arduino plugged into the powerbank.
