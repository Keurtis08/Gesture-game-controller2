Arduino Nano 33 BLE Sense

The brain. Its built-in IMU sensor captures hand movement. An Edge Impulse AI model classifies the gesture in milliseconds and broadcasts it via Bluetooth BLE. Powered by a powerbank — fully wireless.

📡

Bluetooth BLE

The Arduino connects directly to the PC's built-in Bluetooth. No extra hardware needed. The gesture label (UP, DOWN, LEFT, RIGHT) is transmitted wirelessly in real time.

🐍

Python Script

Running on the PC, it receives gesture labels via BLE using bleak and simulates the matching arrow key press with pynput. The game receives it like a real keyboard.
