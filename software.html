Software

The Arduino sketch and the Python script that power the system.

Arduino Sketch — Nano 33 BLE Sense

This code runs on the Arduino. It reads the IMU sensor, classifies the gesture with the Edge Impulse AI model, and sends the label wirelessly via BLE.

/*
 * Gesture Game Controller — Arduino Nano 33 BLE Sense
 * Arduino Days 2026 @ Semè City
 *
 * Reads hand movement from the built-in IMU,
 * classifies the gesture using Edge Impulse AI,
 * and broadcasts the result via Bluetooth BLE.
 */

#include <ArduinoBLE.h>
#include <Arduino_LSM9DS1.h>
// #include <your_model_inferencing.h>  ← replace with your EI model

#define BLE_DEVICE_NAME       "GestureController"
#define CONFIDENCE_THRESHOLD   0.70

BLEService gestureService("180C");
BLEStringCharacteristic gestureChar("2A56", BLERead | BLENotify, 10);

void setup() {
  Serial.begin(115200);
  pinMode(LED_BUILTIN, OUTPUT);

  if (!IMU.begin()) { Serial.println("IMU error!"); while (1); }
  if (!BLE.begin())  { Serial.println("BLE error!");  while (1); }

  BLE.setLocalName(BLE_DEVICE_NAME);
  BLE.setAdvertisedService(gestureService);
  gestureService.addCharacteristic(gestureChar);
  BLE.addService(gestureService);
  BLE.advertise();

  Serial.println("Arduino ready — advertising as: GestureController");
}

void loop() {
  BLEDevice central = BLE.central();

  if (central) {
    Serial.print("PC connected: ");
    Serial.println(central.address());
    digitalWrite(LED_BUILTIN, HIGH);  // LED ON = connected

    while (central.connected()) {
      float ax, ay, az;
      if (IMU.accelerationAvailable()) {
        IMU.readAcceleration(ax, ay, az);

        // Replace this with your Edge Impulse inference call
        String gesture = classifyGesture(ax, ay, az);

        if (gesture != "") {
          gestureChar.writeValue(gesture);
          Serial.print("Gesture sent: ");
          Serial.println(gesture);
          // Flash LED on gesture
          digitalWrite(LED_BUILTIN, LOW);
          delay(80);
          digitalWrite(LED_BUILTIN, HIGH);
        }
      }
      delay(50);
    }
    digitalWrite(LED_BUILTIN, LOW);
    Serial.println("PC disconnected.");
  }
}

// Simplified tilt detection — replace with Edge Impulse model
String classifyGesture(float ax, float ay, float az) {
  if (ay >  0.6) return "UP";
  if (ay < -0.6) return "DOWN";
  if (ax < -0.6) return "LEFT";
  if (ax >  0.6) return "RIGHT";
  return "";
}

Key concepts

📛

BLE device name

GestureController is the name the Arduino broadcasts. The Python script searches for exactly this name. Both must match.

🔔

BLENotify

The PC is automatically notified every time a new gesture is written — no need to constantly poll for updates.

💡

LED feedback

LED stays ON while the PC is connected. It flashes briefly each time a gesture is successfully detected.

Python Script

This script runs on the PC. It connects to the Arduino via Bluetooth BLE, receives gesture labels wirelessly, and simulates the matching arrow key press.

"""
Gesture Game Controller — Python Script
Arduino Days 2026 @ Semè City

Connects to Arduino via Bluetooth BLE,
receives gesture labels, simulates arrow key presses.
"""

import asyncio
from bleak import BleakScanner, BleakClient
from pynput.keyboard import Key, Controller

# Must match the Arduino BLE device name exactly
BLE_DEVICE_NAME   = "GestureController"
GESTURE_CHAR_UUID = "00002a56-0000-1000-8000-00805f9b34fb"

GESTURE_MAP = {
    "UP":    Key.up,
    "DOWN":  Key.down,
    "LEFT":  Key.left,
    "RIGHT": Key.right,
}

keyboard = Controller()

def simulate_key(gesture: str):
    """Press and release the arrow key matching the gesture."""
    key = GESTURE_MAP.get(gesture.strip())
    if key:
        keyboard.press(key)
        asyncio.get_event_loop().call_later(0.05, keyboard.release, key)
        print(f"Key pressed: {gesture} → {key}")

def on_gesture_received(sender, data: bytearray):
    """Called automatically every time the Arduino sends a gesture."""
    gesture = data.decode("utf-8").strip()
    if gesture in GESTURE_MAP:
        simulate_key(gesture)

async def main():
    print(f"Scanning for '{BLE_DEVICE_NAME}' via Bluetooth...")
    device = await BleakScanner.find_device_by_name(BLE_DEVICE_NAME)
    if device is None:
        print("ERROR: Arduino not found.")
        print("Make sure the Arduino is powered on and Bluetooth is ON.")
        return

    print(f"Found! Connecting to {device.address}...")
    async with BleakClient(device) as client:
        print("Connected! Waiting for gestures...\n")
        await client.start_notify(GESTURE_CHAR_UUID, on_gesture_received)
        while client.is_connected:
            await asyncio.sleep(0.1)

if __name__ == "__main__":
    asyncio.run(main())

How to run

1

Install dependencies

pip install bleak pynput
2

Enable Bluetooth on your PC

Make sure Bluetooth is turned ON in your PC settings before running the script.

3

Power on the Arduino

Plug the Arduino into the powerbank. The LED lights up when the PC connects.

4

Run the script

python gesture_controller.py
5

Open the game and play!

Open any arrow-key game (Beach Buggy Racing, etc.) and move your hand. 🎮
